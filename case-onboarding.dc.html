Getting new users set up quickly & confidently

Helping new users get started with 1Password quickly, with clear guidance and confidence from day one. Two efforts: a guided onboarding experience and a ground-up rebuild of sign-up for individuals and organizations.

// discovery

What users told us, and what competitors did well

Customer themes
  • Setup feels difficult and intimidating
  • "I'm definitely not using this app the right way"
  • Advanced features feel hidden or underutilized
Competitive insights
  • Desktop onboarding is more comprehensive and intentional
  • Progress indicators help motivate completion
  • 1Password's onboarding felt fragmented, not holistic
// design

Designing a guided, goal-oriented setup

Defining essential setup steps

Through multiple iterations we determined which tasks truly mattered during onboarding, a set aligned across the design team and scoped to give users the context they need to succeed.

Keeping users motivated through multi-step setup

Progress indicators, clear milestones, and positive reinforcement maintain momentum. Users always know where they are, what's left, and can skip optional steps or return later without losing progress.

Expanding to B2B

Not all organizations want employees completing the same tasks. An admin-controlled setup page lets teams tailor onboarding steps to align with company policies.

// design

A durable, product-led visual system

Previous attempts mirrored the marketing site and fell out of sync as it evolved. We established a standardized, product-led visual system, more durable, easier to maintain, and refined over time.

Native mobile onboarding

Sign-up previously relied on a shared web experience across platforms. Moving to a fully native mobile flow gave us end-to-end control and a smoother, more intuitive setup tailored to mobile users.
